# ApiSvr

ApiSvr 是一个用于管理 API 的工具，它可以帮助你快速创建、管理和部署 API。

## 功能特点

- 快速创建 API
- 简单的配置管理
- 支持多种部署方式

## 快速开始

```bash
cargo install apisvr
```
